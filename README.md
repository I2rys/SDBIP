## Installation
Github:

    git clone https://github.com/I2rys/SDBIP
    
Python Packages:

    pip install -r requirements.txt

## Setup
1. Open index.py
2. Find DiscordBotTokenHere and replace it with your Discord bot token.

## Usage

    python index.py
